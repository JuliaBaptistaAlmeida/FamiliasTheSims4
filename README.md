# 🏡 FamiliaTheSims4

✨ Apresento um dos projetos mais divertidos que já fiz!

Este programa gera automaticamente os integrantes de uma família para **The Sims 4**, ajudando a deixar a gameplay mais variada, imprevisível e divertida.

Em vez de sempre criar personagens manualmente, o programa decide características da família e traz aquele elemento clássico de surpresa — quase como rolar um dado em um RPG. 🎲

## 🎮 Ideia do Projeto

Ao iniciar o programa, ele gera uma família com diferentes características, criando novas possibilidades para jogar **The Sims 4**.

Isso ajuda a:

- Criar famílias diferentes a cada execução;
- Deixar a gameplay mais criativa;
- Sair da zona de conforto ao jogar!

Tenho usado muito na hora de preencher as minhas cidades 💕
